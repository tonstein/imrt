Detailed Description

This class fills the skeleton given by a parameter layout with life. It has a parameter value to which changes can be announced via DspParameter::announce(). This is typically done by the GUI thread. The changes are applied when the DspParameter::updatedValue() method is called. This is typically done by the DSP thread.
